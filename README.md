### CRCCalc
CRC calculator

### Feature

#### Support CRC calculator
- [ ] CRC8
- [x] CRC16
- [ ] CRC32

#### Custom parameter
- [x] Polynomial [CRC standards and common use](https://en.wikipedia.org/wiki/Cyclic_redundancy_check#Standards_and_common_use)
- [x] Initial Value
- [ ] Final Xor Value
- [ ] Input reflected
- [x] Result reflected

#### More feature
- Auto check and update new version
- Simple to use
- Flexible hex input
